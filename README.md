# 🚲 《[自行车后座的风](https://lonliaow.github.io/)》

老城区的梧桐树下，总能看见一个穿着蓝色校服的女孩，坐在一辆二八自行车的后座上，手里举着一支草莓味的冰淇淋，笑容像阳光一样灿烂。男孩穿着白色的衬衫，头发被风吹得有些凌乱，车铃叮铃叮铃地响，穿过铺满落叶的柏油路。

他们是同班同学，男孩叫阿阳，女孩叫小雨。阿阳每天都会骑自行车送小雨回家，从学校到小雨家有三条路，但他总是会绕最远的那一条，因为那条路的梧桐树下有一家冰淇淋店，小雨最喜欢吃那里的草莓味冰淇淋。

他们会在放学路上分享一包干脆面，阿阳吃原味的，小雨吃番茄味的。有时阿阳会故意骑得很快，让小雨的头发被风吹得飘起来，然后笑着说“你像一只小蝴蝶”。小雨会假装生气地捶他的后背，却忍不住笑出声。

上个月学校举办运动会，阿阳参加了1500米长跑，小雨在终点线等着他，手里拿着一瓶矿泉水。阿阳冲过终点线时，一下子扑进她的怀里，汗水浸湿了他的衬衫，他的呼吸很急促，说“我赢了”。小雨帮他擦去额头上的汗水，指尖碰到他的脸颊时，他的脸一下子红了，像熟透的桃子。

昨天我在梧桐树下看见他们，阿阳还是骑着那辆二八自行车，小雨坐在后座上，手里举着一支草莓味的冰淇淋，只是他们都长高了，校服也换成了高中的款式。阳光透过梧桐树叶的缝隙洒下来，落在他们身上，像一幅温暖的画。

现在他们还是每天一起上下学，自行车后座的风里，总带着草莓冰淇淋的甜香味，像他们的青春一样，充满了温暖和希望。

金句： 爱情是自行车后座的风，带着青春的气息，吹过我们的年少时光，留下最美好的回忆。
